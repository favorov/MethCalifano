Which Vista enhancers are differentially methylated in the 2014-10-07_DiscoveryCohort_ClinicalData-Final_DeID.csv data?

Alexander Favorov

07 Dec 2014

We load the Vista Enhancers list from UCSC DAS server (2014-12-02 20:23:37) and we load our MACS as described in 2014-10-07_DiscoveryCohort_ClinicalData-Final_DeID.csv file.

Then, for each island we calculate the net length of methylated regions (we read them from to MACS bedfiles) that ovelap with the island in each of the samples.

Thus, for each island we have a methylated length (coverage) for each sample). Then, we calculate the Wilcoxon p-value for each island's tumor and normal DM coverages. We think that an island is differentially methylated if the p-value is <=0.05. We do the same for Fisher's test.

Here is the list of all differentially methylated (p<=0.05) CpG with Bonferroni correction

In the table: The DM CpG islands coords and the methylation status description; Fisher's test results cytoband information; TSS near is the closest start of a HGNC gene (according to Ensemble coords as represented in Homo.sapiens and org.Hs.eg.db) to the island, position and strand are of the gene. Distance is the distance from the middle of the island to TSS, negative means that the island is in the upstream. We show Wilcoxon and Fisher's p-values for each island as well as OR and the CI for OR. We also show all the HGNC's genes TSS that ovelap the island +- 7000 bp.
