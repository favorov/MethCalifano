Which CpG islands are differentially methylated in the 2014-02-06_RO1_Batch1-4_ClinicalData_DeID data?

Alexander Favorov

08 Apr 2014

We load the CpG islands list from UCSC DAS server and we load our MACS as described in 2014-02-06_RO1_Batch1-4_ClinicalData_DeID.xls.csv file.

Then, for each island we calculate the net length of methylated regions (we read them from to MACS bedfiles) that ovelap with the island in each of the samples.

Thus, for each island we have a methylated length (coverage) for each sample). Then, we calculate the Wilcoxon p-value for each island's tumor and normal DM coverages. We think that am island is differentyally methylated if the p-value is <=0.05.
