Which cytobands are differentially methylated in the 2014-10-07_DiscoveryCohort_ClinicalData-Final_DeID.csv data?

Alexander Favorov

11 Nov 2014

We load the cytoband list (karyotype) from the SNPchip, version 2.12.0 we load our MACS as described in 2014-10-07_DiscoveryCohort_ClinicalData-Final_DeID.csv file.

Then, for each cytoband we calculate the net length of methylated regions (we read them from to MACS bedfiles) that ovelap with the cytoband in each of the samples.

Thus, for each interval (cytodand) we have a methylated length (coverage) for each sample). Then, we calculate the Wilcoxon p-value for each cytoband's tumor and normal DM coverages. We think that a cytoband is differentyally methylated if the p-value is <=0.05.
